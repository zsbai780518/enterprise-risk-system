### 新增 ✨
- **用户系统**
  - 手机号注册/登录
  - 用户名注册/登录
  - 微信快捷登录（预留）
  - 个人中心管理

- **企业管理**
  - 企业搜索（名称/统一社会信用代码）
  - 企业详情展示
  - 企业绑定/解绑
  - 数据采集（手动/批量）

- **风险分析**
  - 风险评分算法
  - 风险等级评估（低/中/高）
  - 风险分布雷达图
  - 关键风险点展示
  - 资质问题检测
  - 智能建议生成

- **可视化图表**
  - 股权结构图
  - 风险地图（雷达图/饼图）
  - 资质合规图谱
  - Canvas 原生绘图

- **报告系统**
  - 风险分析报告生成
  - 报告列表管理
  - 报告下载/分享

- **后端 API**
  - ThinkPHP 6.x/8.x 框架
  - RESTful API 设计
  - JWT Token 认证
  - 10 张核心数据表

- **前端应用**
  - UniAPP 多端适配
  - Vue 3 组合式 API
  - 11 个核心页面
  - 响应式 UI 设计

### 技术栈 🛠️
- 后端：ThinkPHP 6.x/8.x + PHP 7.2.5+
- 前端：UniAPP + Vue 3
- 数据库：MySQL 5.7+
- 可视化：Canvas + ECharts
- 服务器：LAMP (Linux + Apache + MySQL + PHP)

## 版本说明

### 语义化版本

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新频率

- 严重 bug 修复：随时发布热修复
- 功能更新：每月发布次版本
- 重大更新：每季度发布主版本
