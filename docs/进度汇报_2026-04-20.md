## 🚀 如何部署测试

### 方式一：使用部署脚本（推荐）

```bash
cd /home/admin/openclaw/workspace/enterprise-risk-system
bash scripts/deploy.sh
```

脚本会自动：
1. 检查 PHP/MySQL 环境
2. 创建数据库并导入表结构
3. 配置后端
4. 安装 Composer 依赖

### 方式二：手动部署

1. **创建数据库**
```bash
mysql -u root -p -e "CREATE DATABASE enterprise_risk_system DEFAULT CHARACTER SET utf8mb4;"
mysql -u root -p enterprise_risk_system < database/schema_v1.0.sql
```

2. **配置后端**
编辑 `backend/config/database.php` 填写数据库密码

3. **配置前端**
编辑 `frontend/api/index.js` 修改 `BASE_URL` 为实际 API 地址

4. **部署到服务器**
- 后端：将 `backend/public` 目录设为网站根目录
- 前端：使用 HBuilderX 编译后部署
