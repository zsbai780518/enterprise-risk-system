## 📦 项目概述

企业全维度风险智能分析 SaaS 系统，基于 **UniAPP + ThinkPHP + LAMP** 架构开发。

### 核心功能

- 🔐 **用户系统** - 注册/登录/个人中心
- 🏢 **企业管理** - 搜索/采集/绑定/详情
- 📊 **风险分析** - 评分/等级/可视化图表
- 📋 **报告系统** - 生成/下载/分享
- 📱 **多端适配** - UniAPP 一套代码多端运行

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/zsbai780518/enterprise-risk-system.git
cd enterprise-risk-system
```

### 2. 数据库初始化

```bash
# 创建数据库
mysql -u root -p -e "CREATE DATABASE enterprise_risk_system DEFAULT CHARACTER SET utf8mb4;"

# 导入表结构
mysql -u root -p enterprise_risk_system < database/schema_v1.0.sql
```

### 3. 配置后端

编辑 `backend/config/database.php`:

```php
'hostname' => '127.0.0.1',
'database' => 'enterprise_risk_system',
'username' => 'root',
'password' => 'your_password',
```

### 4. 部署

```bash
# 使用一键部署脚本
bash scripts/deploy.sh
```

### 5. 前端开发

使用 **HBuilderX** 打开 `frontend/` 目录，运行到微信小程序/H5/App。

## 📡 API 接口

### 用户相关
- `POST /api/user/register` - 用户注册
- `POST /api/user/login` - 用户登录
- `GET /api/user/info` - 用户信息

### 企业相关
- `GET /api/company/list` - 企业列表
- `POST /api/company/search` - 搜索企业
- `GET /api/company/detail` - 企业详情

### 风险分析
- `GET /api/risk/analysis` - 风险分析
- `GET /api/risk/list` - 风险列表
- `GET /api/chart/equity` - 股权结构图

完整 API 文档见 `docs/API.md`（待完善）
